ngeo

JavaScript lib combining OpenLayers 3 and AngularJS.

  • For people developping web mapping apps with OpenLayers 3 and Angular
  • OpenLayers 3: Open-source JS lib for adding maps to web page
  • AngularJS: Open-source JS framework created by Google
  • Goal is to ease the development
  • ngeo is open-source

Toolkit library to create rich map web applications.

Features

Examples of components provided by ngeo:

  • Layer Tree
  • Measure Tools
  • Elevation Profile
  • …

  • Examples shown later

Design principle

Provide atomic components that the app developer can combine in different ways.

Don't be intrusive in your main application.

Get rid of UI aspects.

  • Other ol3/ng lib exist, why our is unique and could be usefull ?
  • We needed to take design decision on what we wanted ngeo to be
  • Modularity
  • The goal is not to provide pre-packaged widgets

Declarative versus Imperative
