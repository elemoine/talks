ngeo

JavaScript toolkit combining OpenLayers 3 and AngularJS, aimed at creating rich web-mapping applications

  • For developers who use OpenLayers and Angular for creating web-mapping apps
  • OpenLayers 3: open-source JS lib for adding maps to web page
  • Angular: open-source JS framework created by Google
  • Goal is to ease development
  • ngeo is open-source

Features

Targeted features include:

  • Layer tree
  • Measure tools
  • Elevation profile
  • …

  • ngeo provides both low-level and high-level components
  • Examples shown later

Design principle

Provide atomic components that can be combined in many different ways

  • ngeo should not get in your way
  • ngeo should constrain you
  • for example ngeo should not limit you in the way you can use OpenLayers 3
  • other ol3/ng libs exist, what does make ngeo unique?
  • flexibility, modularity
  • the primary goal is not to provide out-of-the-box widgets
  • you can style your components the way you want
  • ngeo is not tied to a specific UI toolkit (although we use it mainly with Bootstrap)
