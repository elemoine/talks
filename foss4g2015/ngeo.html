ngeo

JavaScript lib combining OpenLayers 3 and AngularJS.

  • For people developping web mapping apps with OpenLayers 3 and Angular
  • OpenLayers 3: Open-source JS lib for adding maps to web page
  • AngularJS: Open-source JS framework created by Google
  • Goal is to ease the development
  • ngeo is open-source

Features

Examples of components provided by ngeo:

  • Layer Tree
  • Measure Tools
  • Elevation Profile
  • …

  • Examples shown later

Design principle

Provide atomic components that the app developer can combine in different ways.
